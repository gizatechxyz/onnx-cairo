# int.abs 

```rust
fn abs(self: T) -> T;
```

Computes the absolute value of a signed\_integer.

## Args

`self`(`T`) - The signed integer to which the absolute value is applied

## Returns

A signed integer `<T>`, representing the absolute value of `self` .
